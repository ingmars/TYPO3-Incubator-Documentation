Making the Web Accessible
#########################

Much of the focus on Web accessibility has been on the responsibilities of Web
developers. However, Web software, as TYPO3, also has a vital role in Web
accessibility. Software needs to help developers produce and evaluate
accessible Web sites, and be usable by people with disabilities.
