TYPO3 Editor
############

With about 25% of Internet users needing accessibility in some way, there
are very clear reasons to make your website accessible. All users need to
have equal access to information and functionality.

The TYPO3 software, a website integrator, and an extension developer all play
roles in designing and developing a site that everyone can view and understand,
but it doesn't stop there. As an editor you have an important role in creating
and accessible website!

Some of the basic :doc:`../../1-MakingWebAccessible/2-principles` to make a
website accessible do apply to editors, but there are some TYPO3-specific tips
as well.
